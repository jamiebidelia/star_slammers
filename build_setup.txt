Get together the required software and libraries:
sudo apt-get install libncurses-dev
sudo apt-get install rustc
sudo apt-get install lua5.3
sudo apt-get install git

Get your own copy of the codebase:
git clone www.github.com/jamiebidelia/star_slammers

Build the code:
navigate to project directory
cargo build

Build the documentation:
./gen_doc.sh
